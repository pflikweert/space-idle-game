# Game Vision

## Working title

`space-idle-game`

## Concept status

This project is in concept phase. The direction below is a working prototype brief, not locked product truth.

## Core fantasy

- pilot a small ship or early fleet
- move through a calm but dangerous space frontier
- grow from fragile scavenging into confident system-to-system progression

## Style direction

- mobile-first
- readable on web
- pixel / 2.5D inspired presentation
- quiet, atmospheric space scenes with moments of spectacle

## Audience and market idea

This is a provisional idea for players who like approachable idle progression, light strategy, and compact defense decisions on mobile or web. It is not yet validated market truth.

## Provisional loop

`scan -> mine/collect -> upgrade -> defend -> travel`
